## v/dv(dynamic viewport)/sv(small viewport)/lv(large viewport)
### v
- 브라우저 크기
### dv - 특별한 경우
- 실시간으로 변화, 보여주는 뷰포트
### sv - 안정적
- ui 포함된 뷰포트 ex) 중요한 컨텐츠
### lv - 최대 화면 활용
- ui 포함안된 뷰포트
